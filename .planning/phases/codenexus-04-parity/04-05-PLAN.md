```bash
#!/usr/bin/env bash
# Usage:
#   preseed-hf-cache.sh --source <existing-cache-dir> [--target <HF_HOME>] [--revision <sha>] [--model <repo-id>]
#
# Default model: Qwen/Qwen3-Embedding-0.6B
# Default revision: 97b0c614be4d77ee51c0cef4e5f07c00f9eb65b3
# Default target: ${HF_HOME:-$HOME/.cache/huggingface}/hub
#
# Modes:
#   - --source <dir>: copy from another HF cache dir (e.g., shared NAS, Linux/macOS host scp)
#   - --source <tarball>: extract pre-built tarball into target
#   - --verify-only: skip copy, just check target has expected snapshot/<sha>/model.safetensors
```

Implementation:
1. Parse flags (`--source`, `--target`, `--revision`, `--model`, `--verify-only`, `--help`).
2. Resolve target dir: `${HF_HOME:-$HOME/.cache/huggingface}/hub/models--<org>--<name>/`.
3. If `--source` is a tarball (`*.tar.gz` or `*.tar`): extract to target with `tar -xzf` / `tar -xf`.
4. If `--source` is a directory: `cp -rL <src>/snapshots/<sha>/. <target>/snapshots/<sha>/` to resolve symlinks (portable across hosts). Also copy `refs/main` if it exists.
5. Verify target after copy: check `target/snapshots/<sha>/model.safetensors` exists, size > 1GB, sha256 matches if available (manifest file in source).
6. Print summary line: `[preseed] target=<dir> sha=<sha> safetensors=<size>MB OK`.
7. Exit 0 on success, non-zero with clear error message on failure.

Cross-platform notes:
- Bash on git-bash (Windows) and Unix.
- Avoid POSIX-incompat features (no `mapfile`, no `<<<`).
- Use `/usr/bin/cat` if heredoc embeds (per [P0] heredoc-bash-cat-fullpath rule, `cat` is aliased to `bat` on this host).

Create `experiments/poc-retrieval/eval/r1c_probe.sh` (~70 lines, mirrors `r4b_probe.sh` style):

```bash
#!/usr/bin/env bash
# r1c_probe.sh -- standalone R1.c reload probe (pre-seed flavor).
#
# Original R1.c spec: delete snapshot dir + redownload yields same SHA.
# Pre-seed flavor: delete target snapshot dir + run preseed-hf-cache.sh
# from a known source + run codenexus eval + assert byte-identical results
# vs. baseline. Proves SHA pin survives a full pre-seed cycle.

set -euo pipefail
cd "$(dirname "$0")/.."

BIN="./target/release/codenexus-core.exe"
DB="./poc.db"
SHA="97b0c614be4d77ee51c0cef4e5f07c00f9eb65b3"
SOURCE_CACHE="$HOME/.cache/huggingface"   # use existing cache as the source
TEST_HF_HOME="$(mktemp -d -t codenexus-r1c-XXXXXX)"
LOG_DIR="./eval/r1c_logs"
mkdir -p "$LOG_DIR"

trap 'rm -rf "$TEST_HF_HOME"' EXIT

# Phase 1: pre-seed into isolated HF_HOME
HF_HOME="$TEST_HF_HOME" bash scripts/preseed-hf-cache.sh \
  --source "$SOURCE_CACHE/hub/models--Qwen--Qwen3-Embedding-0.6B" \
  --target "$TEST_HF_HOME/hub/models--Qwen--Qwen3-Embedding-0.6B" \
  > "$LOG_DIR/preseed1.log" 2>&1

# Run eval, capture mean precision_at_5
HF_HOME="$TEST_HF_HOME" HF_HUB_OFFLINE=1 "$BIN" eval \
  --queries eval/queries.json --db "$DB" --alpha 0.6 \
  --out "$LOG_DIR/eval1.json" > "$LOG_DIR/eval1.log" 2>&1

# Phase 2: nuke snapshot dir, re-pre-seed
rm -rf "$TEST_HF_HOME/hub/models--Qwen--Qwen3-Embedding-0.6B/snapshots/$SHA"
HF_HOME="$TEST_HF_HOME" bash scripts/preseed-hf-cache.sh \
  --source "$SOURCE_CACHE/hub/models--Qwen--Qwen3-Embedding-0.6B" \
  --target "$TEST_HF_HOME/hub/models--Qwen--Qwen3-Embedding-0.6B" \
  > "$LOG_DIR/preseed2.log" 2>&1

# Re-run eval
HF_HOME="$TEST_HF_HOME" HF_HUB_OFFLINE=1 "$BIN" eval \
  --queries eval/queries.json --db "$DB" --alpha 0.6 \
  --out "$LOG_DIR/eval2.json" > "$LOG_DIR/eval2.log" 2>&1

# Assertion: byte-identical (deterministic equality, not +/-2pp)
if diff -q "$LOG_DIR/eval1.json" "$LOG_DIR/eval2.json" > /dev/null; then
  echo "PASS: R1.c -- pre-seed reload yields byte-identical eval results"
  exit 0
else
  echo "FAIL: R1.c -- eval drift detected after re-preseed"
  diff "$LOG_DIR/eval1.json" "$LOG_DIR/eval2.json" | head -20
  exit 1
fi
```
</action>
