<objective>
The behavior-critical wave. Rewire `core/src/graph_build.rs` to consume the
parser sub-crate primitives (PathResolver / CallResolver / AliasIndex /
SymbolIndex / RepoCtx) instead of the inline tree-sitter queries +
hand-rolled resolver methods. This is the only wave that materially changes
the runtime indexing path. T1-T7 (commit `4ebffa9` characterization tests)
are the safety net -- they MUST stay green AGAINST THE NEW CODE PATH.

Key transformations:
1. **Replace 4 inline tree-sitter query consts** (Q_CALLS / Q_IMPORTS /
   Q_IMPLEMENTS / Q_EXTENDS at graph_build.rs lines 38-94) with calls into
   the parser sub-crate. Q_IMPORTS becomes a call to
   `parser::imports::extract_aliases(file, &repo_ctx) -> Vec<AliasDecl>`.
   Q_CALLS / Q_IMPLEMENTS / Q_EXTENDS stay AS TREE-SITTER QUERIES at this
   stage but become consumers of the parser-emitted FoundEdge stream
   produced by an ast_import_walker integration. (Actual TS calls walker
   ships in W4 -- W3 keeps the existing query logic for Calls / Implements
   / Extends and ONLY swaps Imports + the resolver tail.)

2. **Replace `resolve_with_conf` + `resolve_with_namespace` methods**
   (graph_build.rs lines 437-484) with a call to
   `CallResolver::resolve(file, name, ns_obj, &symbol_index, &alias_index)
   -> Option<(i64, ResolutionMethod)>`. The f64 confidence derives via
   `confidence_from_method` at the call to `Store::insert_edge_conf`.

3. **Replace `resolve_import_path`** (graph_build.rs lines 487-536) with
   `PathResolver::resolve(from_file, source) -> Option<String>`.

4. **Switch Imports rows storage from `edges` table to `alias_decls` table.**
   The W0 schema has alias_decls ready; W3 actually starts writing to it.
   PASS 1 (`pass_imports`) becomes "extract AliasDecls + persist via
   `Store::insert_alias_decl`". The `Imports` enum variant in the runtime
   code is removed; `EdgeKind` runtime values reduce to Calls / Implements
   / Extends. The SQL CHECK constraint on `edges.kind` is TIGHTENED in this
   wave (W0 left it permissive; W3 removes 'Imports' from the CHECK list).

5. **Two-pass dependency order preserved.** The ALD (AliasDecl) pass runs
   before the Calls/Implements/Extends pass because CallResolver needs a
   populated AliasIndex. graph_build.rs::build_all becomes:
   ```
   PASS 1: for each file, extract AliasDecls + insert into alias_decls table
   PASS 2: build SymbolIndex once; build AliasIndex once from list_alias_decls(*)
   PASS 3: for each file, walk Calls/Implements/Extends queries; resolve via CallResolver
   ```

6. **Delete `core/src/parser.rs`** (the OLD codenexus parser with the
   Language enum + dispatch). Symbol already moved out of
   `core::parser::Symbol` in W1 Task 0 (extracted to `codenexus_types::Symbol`
   leaf crate); W1 left a re-export shim (`pub use codenexus_types::Symbol;`)
   in core/src/parser.rs for source-compat. W3 now deletes the file entirely.
   Other public types in parser.rs (Language enum, detect_language, LangCtx,
   etc.) migrate either to parser sub-crate or get inlined into the call
   sites that need them -- the executor enumerates these on read.

7. **Symbol parsing** (the OLD parser.rs's reason-to-exist) is delegated to
   the parser sub-crate via `parser::ast_import_walker` + a new top-level
   `parser::extract_symbols(file, source, &repo_ctx) -> Vec<Symbol>`.
   The lang_extractor framework lifted in W1 already has this -- W3
   wires graph_build.rs::pass_imports' "parse this file" step to it.

```rust
pub fn build_all(&mut self) -> Result<EdgeStats> {
    let t0 = Instant::now();
    let mut stats = EdgeStats::default();
    self.storage.clear_edges()?;
    self.storage.clear_alias_decls()?;     // W3: idempotent re-index also clears alias_decls

    let files = self.storage.list_files()?;
    eprintln!("graph_build: {} files", files.len());

    // PASS 1: Extract AliasDecls (replaces old pass_imports). For each file,
    // the parser sub-crate's ast_import_walker emits Vec<AliasDecl> from
    // tree-sitter parse output. We persist via Store::insert_alias_decl.
    for f in &files {
        if let Err(e) = self.pass_alias_decls(f, &mut stats) {
            eprintln!("alias_decls error in {}: {}", f, e);
        }
    }
    eprintln!("pass1 alias_decls: {} aliases recorded; unresolved imports {}",
              stats.imports, stats.unresolved);

    // PASS 2: Build SymbolIndex once (Store -> projection) + AliasIndex once
    // (alias_decls table -> projection). These cache per-call-site lookups
    // that the pre-04.5-03 code did via SQL on every call site.
    let symbol_index = codenexus_parser::SymbolIndex::from_store(self.storage)?;
    let mut all_decls = Vec::new();
    for f in &files {
        all_decls.extend(self.storage.list_alias_decls(f)?);

    let alias_index = codenexus_parser::AliasIndex::from_decls(&all_decls);

    // PASS 3: Resolve Calls / Implements / Extends edges via CallResolver.
    for f in &files {
        if let Err(e) = self.pass_relations_v2(f, &symbol_index, &alias_index, &mut stats) {
            eprintln!("relations error in {}: {}", f, e);
        }
    }

    eprintln!("graph_build done in {} ms", t0.elapsed().as_millis());
    Ok(stats)
}
```

```rust
fn pass_alias_decls(&mut self, file: &str, stats: &mut EdgeStats) -> Result<()> {
    let src = match self.read_source(file) {
        Some(s) => s,
        None => return Ok(()),

    // The parser sub-crate's imports module exposes the equivalent of
    // sentrux's parser/imports.rs walk. The exact public function name was

    for a in aliases {
        // The walker may emit AliasDecls with target_file already resolved
        // by PathResolver. If the walker emits unresolved sources, run
```

```rust
fn pass_relations_v2(
    &mut self,
    file: &str,
    symbol_index: &codenexus_parser::SymbolIndex,
    alias_index: &codenexus_parser::AliasIndex,
    stats: &mut EdgeStats,
) -> Result<()> {
    let src = match self.read_source(file) {
        Some(s) => s,
        None => return Ok(()),
    };
    let tree = match self.parser.parse(&src, None) {
        Some(t) => t,
        None => return Ok(()),
    };
    let root = tree.root_node();
    let symbols_in_file = self.storage.symbols_in_file_full(file)?;
    if symbols_in_file.is_empty() {
        return Ok(());
    }

    // Calls — namespace-aware. ns_obj capture flows through to CallResolver.
    {
        let callee_idx = self.q_calls.capture_index_for_name("callee").context("callee capture missing")?;
        let ns_obj_idx = self.q_calls.capture_index_for_name("ns_obj");
        let mut cursor = QueryCursor::new();
        let mut matches = cursor.matches(&self.q_calls, root, src.as_bytes());
        while let Some(m) = matches.next() {
            let mut callee_name: Option<(String, usize)> = None;
            let mut ns_obj_name: Option<String> = None;
            for cap in m.captures {
                if cap.index == callee_idx {
                    callee_name = Some((src[cap.node.byte_range()].to_string(), cap.node.start_position().row + 1));
                } else if Some(cap.index) == ns_obj_idx {
                    ns_obj_name = Some(src[cap.node.byte_range()].to_string());
                }
            }
            let (name, row) = match callee_name { Some(t) => t, None => continue };
            let from_id = match enclosing_symbol(&symbols_in_file, row) { Some(id) => id, None => continue };
            if from_id == 0 { continue; }
            let resolved = codenexus_parser::CallResolver::resolve(
                file, &name, ns_obj_name.as_deref(),
                symbol_index, alias_index,
            );
            match resolved {
                Some((to_id, method)) if to_id != from_id => {
                    let conf = codenexus_parser::confidence_from_method(method);
                    self.storage.insert_edge_conf(from_id, to_id, "Calls", conf)?;
                    stats.calls += 1;
                }
                Some(_) => continue,  // self-edge, skip
                None => stats.unresolved += 1,
            }
        }
    }

    // Implements — same shape as Calls but no ns_obj.
    {
        let cap_idx = self.q_implements.capture_index_for_name("impl").context("impl capture missing")?;
        let mut cursor = QueryCursor::new();
        let mut matches = cursor.matches(&self.q_implements, root, src.as_bytes());
        while let Some(m) = matches.next() {
            for cap in m.captures {
                if cap.index != cap_idx { continue; }
                let name = src[cap.node.byte_range()].to_string();
                let row = cap.node.start_position().row + 1;
                let from_id = match enclosing_symbol(&symbols_in_file, row) { Some(id) => id, None => continue };
                if let Some((to_id, method)) = codenexus_parser::CallResolver::resolve(
                    file, &name, None, symbol_index, alias_index,
                ) {
                    if to_id == from_id { continue; }
                    let conf = codenexus_parser::confidence_from_method(method);
                    self.storage.insert_edge_conf(from_id, to_id, "Implements", conf)?;
                    stats.implements += 1;
                } else {
                    stats.unresolved += 1;
                }
            }
        }
    }

    // Extends — same as Implements.
    {
        let cap_idx = self.q_extends.capture_index_for_name("ext").context("ext capture missing")?;
        let mut cursor = QueryCursor::new();
        let mut matches = cursor.matches(&self.q_extends, root, src.as_bytes());
        while let Some(m) = matches.next() {
            for cap in m.captures {
                if cap.index != cap_idx { continue; }
                let name = src[cap.node.byte_range()].to_string();
                let row = cap.node.start_position().row + 1;
                let from_id = match enclosing_symbol(&symbols_in_file, row) { Some(id) => id, None => continue };
                if let Some((to_id, method)) = codenexus_parser::CallResolver::resolve(
                    file, &name, None, symbol_index, alias_index,
                ) {
                    if to_id == from_id { continue; }
                    let conf = codenexus_parser::confidence_from_method(method);
                    self.storage.insert_edge_conf(from_id, to_id, "Extends", conf)?;
                    stats.extends += 1;
                } else {
                    stats.unresolved += 1;
                }
            }
        }
    }

    Ok(())
}
```
